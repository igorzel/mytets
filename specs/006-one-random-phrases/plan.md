# Implementation Plan: One Command - Random Phrase Output

**Branch**: `006-one-random-phrases` | **Date**: 2026-04-21 | **Spec**: [spec.md](spec.md)

## Summary

The `mytets one` command will read an embedded `phrases.json` file, select a random phrase, and output it as plain text (default) or JSON (via global `--output json` flag). The phrase selection uses Go's standard `math/rand` library to ensure uniform random distribution. All errors are reported to stderr with exit code 1, maintaining alignment with the project's reliability principle.

## Technical Context

**Language/Version**: Go 1.26.2 (with Go 1.25+ compatibility)  
**Primary Dependencies**: github.com/spf13/cobra (CLI command framework, already present)  
**Storage**: Embedded JSON file (`phrases.json`) compiled into binary  
**Testing**: Go testing package (standard library) + integration testing  
**Target Platform**: Linux, macOS, Windows (single static binary)  
**Project Type**: CLI tool / command-line utility  
**Performance Goals**: <100ms execution time from invocation to output  
**Constraints**: Sub-100ms startup + execution; minimal memory allocation; no external runtime dependencies  
**Scale/Scope**: Single binary; embedded file ~1-10KB; command-line interface
